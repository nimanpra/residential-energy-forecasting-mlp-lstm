# Submission checklist

## Manuscript

- [ ] Abstract within journal/arXiv limits
- [ ] All figures and tables numbered with captions
- [ ] References consistent (IEEE style in `paper.tex`)
- [ ] Author order and affiliations match everywhere

## arXiv (optional)

- Primary category often used: `cs.LG` (Machine Learning); cross-lists may include `eess.SY`, `stat.ML`
- Upload LaTeX bundle: `arxiv_submission.tar.gz`
- First-time submitters may need [endorsement](https://info.arxiv.org/help/endorsement.html)

## Repository

- [ ] `results.json` matches the paper tables (after re-running `run_experiments.py` if you change code)
- [ ] `data/README.md` reflects any change in data use or attribution

## Keywords (search)

`residential energy forecasting`, `LSTM`, `MLP`, `smart meter`, `Melbourne`, `Bureau of Meteorology`, `solar PV`, `short-term load forecasting`, `time series`
